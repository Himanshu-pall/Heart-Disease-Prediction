# ❤️ Heart Disease Prediction App

A Machine Learning web application built using Python, Streamlit, and Scikit-learn to predict the likelihood of heart disease.

## 🚀 Live Demo
https://heart-disease-prediction-nkg53mz4jnhekqpfmneolo.streamlit.app/

## 📂 Repository
https://github.com/Himanshu-pall/Heart-Disease-Prediction

## 📌 Features
- Predicts the risk of heart disease
- Interactive web interface
- Real-time prediction using a trained KNN model

## 🛠️ Technologies Used
- Python
- Streamlit
- NumPy
- Pandas
- Scikit-learn
- Joblib

## ▶️ Run Locally

bash
git clone https://github.com/Himanshu-pall/Heart-Disease-Prediction.git
cd Heart-Disease-Prediction
pip install -r requirements.txt
streamlit run app.py
